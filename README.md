### When using Microsoft IME to input Chinese in Chrome and Edge on Windows 11, it triggers the web mouse leave event.

## Setup & Run

### Install Node.js

> Node.js version 20.19+ or 22.12+.

Download from https://nodejs.org (LTS version)

> Verify installation:
> ```bash
> node -v
> npm -v
> ```

### Install & Run

```bash
# Install dependencies
npm install

# Start dev server
npm run dev
```

Open http://localhost:5173 in your browser.
